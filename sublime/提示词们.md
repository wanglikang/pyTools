# 功能要求
1、本插件的核心功能，是将待处理文件中的内容，转换为json格式的字符串。这个文件中的内容，可以是以下几种类型:
- 普通的json格式的字符串
- 带有转义符的json格式的字符串
- 带有嵌套结构的json格式的字符串
  
2、针对上面不同的内容类型，注册不同的快捷键，以便于快速转换。
3、转换后的字符串，会自动粘贴到剪贴板中。

# 示例数据
## 示例1 
对应【普通的json格式的字符串】 这种情况

输入：
```plain
{
    "name": "张三",
    "age": 18,
    "sex": "男"
}
```

输出：
```json
{
    "name": "张三",
    "age": 18,
    "sex": "男"
}   
```


## 示例2
对应【带有转义符的json格式的字符串】 这种情况

输入：
```plain
{\"name\": \"张三\",\"age\": 18,\"sex\": \"男\"}
```

输出：
```json
{
    "name": "张三",
    "age": 18,
    "sex": "男"
}   
```


## 示例3
对应【带有嵌套结构的json格式的字符串】 这种情况

输入：
```plain
{
    "name": "张三",
    "age": 18,
    "sex": "男",
    "otherInfo":"{\"name\": \"张三\",\"age\": 18,\"sex\": \"男\"}"
}


```

输出：
```json
{
    "name": "张三",
    "age": 18,
    "sex": "男",
    "otherInfo":
    {
        "name": "张三",
        "age": 18,
        "sex": "男"
    }
}
```
